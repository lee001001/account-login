# 簡單帳號登入畫面
使建立使用者資料已經建立， 帳號密碼符合者可以登入畫面

## 功能
* 根據已設定好的使用者信箱與密碼，可進入使用者的頁面
* 帳號密碼檢查
* 登出即可以消除cookie

## 環境建置與需求
* express v4.17.1
* express-handlebars v5.1.0
* body-parser v1.19.0 
* cookie-parser v^1.4.5 

## 安裝與使用

### 下載專案
```
 git clone https://github.com/lee001001/account-login.git
 ```

### 安裝package
```
npm install
```
### 使用nodemon 啟動
```
npm run dev
```

### 一般啟動
```
npm start
```
